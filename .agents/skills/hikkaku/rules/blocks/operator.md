# Operators

## add(a, b)

Addition.

## subtract(a, b)

Subtraction.

## multiply(a, b)

Multiplication.

## divide(a, b)

Division.

## lt(a, b)

Less-than comparison.

## equals(a, b)

Equality comparison.

## gt(a, b)

Greater-than comparison.

## and(a, b)

Logical AND.

## or(a, b)

Logical OR.

## not(operand)

Logical NOT.

## random(from, to)

Random number.

## join(a, b)

String concatenation.

## letterOf(letter, text)

Character extraction.

## length(text)

String length.

## contains(text, substring)

Substring check.

## mod(a, b)

Modulo.

## round(value)

Rounds number.

## mathop(operator, value)

Math operation (sin, cos, log, etc.).
