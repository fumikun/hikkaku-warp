# Sound

## playSound(sound)

Plays sound.

## playSoundUntilDone(sound)

Plays and waits.

## stopAllSounds()

Stops all sounds.

## setSoundEffectTo(effect, value)

Sets sound effect.

## changeSoundEffectBy(effect, value)

Changes sound effect.

## clearEffects()

Clears sound effects.

## setVolumeTo(value)

Sets volume.

## changeVolumeBy(value)

Changes volume.

## getVolume()

Returns volume.
